Helpful Resources

Python Programming

  • 100 Days of Code: The Complete Python Pro Bootcamp for 2022 teaches Python from the ground up and covers a lot of interesting concepts in an easy to follow format.
  • OnlineGDB allows you to run Python code without any installation.
  • Advent of Code provides programming challenges with some personality.
  • Project Euler challenges both your programming and mathematical abilities.
  • LeetCode contains a massive amount of exercises closely related to technical coding interviews.


Data Science and Machine Learning

  • Complete Machine Learning & Data Science Bootcamp 2022 teaches essential data science and machine learning skills in a repitious way that makes the lessons stick. I prefer PyTorch to Tensorflow, so skip the Tensorflow portion of this course for the bullet point below.
  • PyTorch for Deep Learning teaches the essentials of deep learning using PyTorch. I have yet to work through this course but the instructors previous courses have been so solid I need to initially recommend it. A free sample can be found here. This course will be added to Udemy in the future.
  • Kaggle has thousands of datasets as well as forums and posted solutions. This can be utilized for nearly every type of data science project.
  • Stat Quest presents a wide range of topics in an enjoyable and understandable way.
  • The Hundred-Page Machine Learning Book concisely covers a lot of essential topics in machine learning. The author also has an expanded follow up book Machine Learning Engineering.
